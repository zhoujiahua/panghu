### 官方地址
https://www.videolan.org/vlc/index.zh.html

### 使用方法
 1. 能播放任何内容 - 文件、光盘、摄像头、设备及流媒体
 2. 可播放大多数格式，无需安装编解码器包 - MPEG-2, MPEG-4, H.264, MKV, WebM, WMV, MP3...
 3. 可在所有平台运行 - Windows, Linux, Mac OS X, Unix, iOS, Android ...
 4. 完全免费 - 无间谍软件，无广告，无跟踪用户的行为
